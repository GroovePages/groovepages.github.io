Start with blocks

Get your page on the canvas and looking awesome literally within seconds by using one or more of our pre-designed blocks. Once on the canvas, blocks can be dragged and dropped to change the order in which they appear on your page.

Finishing touches using components

Use our pre-defined components such as navigation bars, maps, headings, text blocks, images, videos and more for a higher level of control. Components already on the canvas can easily be re-arranged by dragging and dropping them.

Easy to use style editor

Modify virtually any style property for any element on the canvas. In addition to setting specific properties, style classes can also be applied, combining multiple styles.

Powerful, yet simple

Additional features

GroovePages is more then just an excellent page and funnel builder

Revision history

GroovePages stores older version of your pages and allows you to revert back to an older version anytime.

Publishing

Allow users to easily publish their sites. Sites can be published in a subfolder, subdomain or custom domain.

Pre-made templates

Create and save full page templates which users can access and use to get their own pages ready faster.

Form handling

Form submissions are easily captured by GroovePages, which will email the data to a specified address.

Image library

Users can upload, resize and crop images in their own library. Users can also access the  general image library.

HTML Editor

Use the GroovePages’ built-in HTML source code editor to gain full control over your page’s source code.
